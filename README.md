## ğŸ“– About

This project is a mobile-first web application designed for healthcare professionals to efficiently record and manage patient data, daily notes, and medical records, utilizing AI to streamline documentation.

## âœ¨ Features

- **Mobile-first Design**: Fully responsive interface for seamless use on any device.
- **AI Integration (Groq)**: Generates professional medical summaries and insights from raw notes.
- **Medication Management**: Dedicated system for tracking and updating patient medications.
- **PDF Generation**: Easily create and export patient records and notes into PDF format.
- **Secure Authentication**: Robust authentication and Row Level Security (RLS) via Supabase.
- **Multi-tenant Architecture**: Supports multiple users while maintaining data segregation.
